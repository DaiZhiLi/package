## Tips
```
1. 深拷贝对象数组 var newArr=[].concat(JSON.parse(JSON.stringify(arr)));

2. stringObject.localeCompare(target)

3. js 数字精度丢失  小数乘以倍数，再缩回原来的倍数

4. ajax请求导致UI进程阻塞  $.Deferred()  $.when().done(）  jquery1.5以上

5. jquery 经典特效交互开发

6. IE8 直接修改type属性报错  修改DOM

7. tracert  ping  ping可以查网速而无法判定出问题的节点，而tracert可以判定出问题的节点，但比较长。
简单来说ping的是查目的的，tracert是查过程的。

8. getTimezoneOffset() 方法可返回格林威治时间和本地时间之间的时差，以分钟为单位。

9. str.replace(/(^\s*)|(\s*$)/g,"") 删除字符串左右两边空格

10. xml
parser=new DOMParser();
xmlDoc=parser.parseFromString(txt,"text/xml");

Internet Explorer 使用 loadXML() 方法来解析 XML 字符串，而其他浏览器使用 DOMParser 对象。

11. ajax   processData
默认值为true,默认情况下data选项传递进来的数据，如果是一个对象都会处理转化成一个查询字符串。// json
如果要发送DOM树信息或其它不希望转换的信息，设置成false。// xml
```
